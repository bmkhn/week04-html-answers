Activity 01

Here are the different tags used in the cozycup-boilerplate:

  • <!DOCTYPE html> - Defines the document type and version of HTML.
  • <html> - Root element of the HTML document.
  • <head> - Contains meta-information about the document.
  • <meta> - Specifies metadata such as character set and viewport settings.
  • <link> - Links to an external stylesheet.
  • <title> - Sets the title of the document.
  • <body> - Contains the content of the HTML document.
  • <header> - Represents introductory content or a set of navigational links.
  • <nav> - Defines a set of navigation links.
  • <div> - Generic container for flow content.
  • <img> - Embeds an image.
  • <ul> - Unordered list.
  • <li> - List item.
  • <section> - Defines a section in the document.
  • <h1> - Heading level 1.
  • <h2> - Heading level 2.
  • <p> - Paragraph.
  • <a> - Hyperlink.
  • <footer> - Represents the footer of a document or section.
